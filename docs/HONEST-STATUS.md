# 誠實現況評估(moo2 remake)

> 起於 2026-07-04 使用者實測回饋「還原度約 20%」。**該 20% 是接原版 LBX 美術前的快照,已被 2026-07-05~12 多輪超越,勿再當現況錨點。** 本文保留核心紀律 + 現況 + 誠實 caveat;逐輪「怎麼接的」細節見 git log 與 `WORKLIST.md`,不在此重複佔篇幅。

## 核心紀律(此文存在的理由,rulebook/65)

還原度**只以「對原版實測比對」驗收,不看單元測試綠 / 新增系統數**。曾多輪用「加自製系統 + 測試綠 + headless 有畫面」謊報進度,但那些係數多是自編近似值(程式碼自標「remake 調校值」),測試只證自製邏輯自洽、不證對齊原版——使用者一玩就見差異。**還原 = 對齊原版,不是長出新東西。誠實標注每個數字來源,自編的就說是自編的。**

## 現況(2026-07-12)

可玩的多帝國 4X 迴圈(headless 探針驗:玩家 + 3 AI 各自擴張、經濟健康、可經三條勝利路徑贏一局、無 panic):

- **① 音樂 ✅**:PCM 場景 BGM(menu/galaxy/diplo/combat)+ 按鈕/戰鬥音效;僅曲目↔場景精確身分待人耳定案。
- **② 忠實新遊戲流程 ✅**:版本選擇 1.3/1.5 → 難度/星系 → 14 族肖像選擇 → 自訂點數 → 命名旗色 → 真母星(人口職務模型農4/工1/科3,SAVE10 校準)。
- **④ gameplay 子系統全接**(非死碼):經濟(建築效果/重力/士氣/礦產/地形改造/污染/成長/收入/指揮點數)、科技抉擇樹、戰鬥(beam/飛彈/球狀分流 + 艦艇佔格 + 武器 mod)、地面戰(陸戰隊/戰車/軌道轟炸)、拓殖、多 AI(N=3,含 AI-vs-AI 關係矩陣)、間諜偷科技、領袖技能、三勝利路徑(征服/議會忠實搖擺票/安塔蘭反攻)。
- **經濟校準**(手冊考據 + SAVE10.GAM oracle + moo2sim 探針):每科研 3、開局 BC 50、人頭基礎收入 1 BC/人、稅率預設 15、指揮基礎供給 5。60 回合玩家 BC 健康爬升(55→419)。詳見 `docs/knowledge-base/original-gameplay-reference.md` §7、`docs/knowledge-base/manual-cht/`。

LBX 解碼(scan-line RLE 影像 / 多幀 delta / 調色盤鏈)、SAVE10.GAM 唯讀解析、16 畫面真原版美術 + 擦底疊字、數百條 UI 譯表、go/ebiten + docker 編譯截圖 + AppImage 打包骨架。

## 仍近似 / 待校準 / oracle-gated(誠實 caveat)

- **③ 按鍵逐畫面像素對齊 ✅ 已做到 oracle 上限**(2026-07-12):逐畫面比對 openorion2 `initWidgets` 真值,把有真值卻用 PIL 的補齊(planets 補第8列、research/fleet/officer/info 座標校正;menu 本就 0px)。無對應 openorion2 view 的畫面(colony/races/newgame/shipDesign)無硬編真值可覆蓋,維持 PIL;結果顯示畫面維持整畫面返回。要再精確只能靠原版截圖(本專案不採)。
- **oracle-gated**:飛彈速度(`missile.go` 手冊公式與附表自相矛盾)、地面戰 d100 核心傷亡解算結構(`ResolveGroundBattle` 沿用一代 1oom 借用結構,force 值用 MOO2 手冊表但結構本身未對 MOO2 實機核實)、安塔蘭母星防禦艦隊戰力(手冊/openorion2 均無精確數字,用保守預設 6 艘末日之星等級)。
- **需先建基礎設施**:戰機/航母(新戰鬥子模型)、部分軍事/防禦建築(~13 棟,需艦隊駐防/軌道防禦系統先落地)。
- **UI 最小化**:議會/安塔蘭無勝利/落敗結局動畫;議會 accept/reject 用文字提示疊在原版底圖,非原版落敗對話框藝術(尚未定位該 LBX);完整 spy/leader/多AI 目標選擇 UI 未做。
- **openorion2 完成度天花板**:MOO2 25 個領袖技能,openorion2 自己也只實作 4 個效果消費端;remake 接了科學家/貿易家 2 個殖民地技能,艦艇軍官技能仍裝飾(非 remake 特有缺口)。
- **turn-1 校準開放項**:科學家分配(科3 vs 原版可能科4)待 playtest 定案。

## 鐵律

驗收看原版實測不看測試綠;不把「新增系統」當還原進度;誠實標數字來源;**還原度 % 待使用者對原版實測定案,不自評**。
