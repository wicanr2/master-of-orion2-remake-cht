## 二、調色盤配對律(關鍵)

大圖(13+)無內嵌調色盤,需借 0–12 其一。**配對律**:

```
種族 r(r = 0..12):
  使節房背景 = asset 13 + 2r
  使節動畫   = asset 14 + 2r
  調色盤源   = asset r          ← 同一個 r
```

**實測驗證**:
- 房 15(r=1)借 palette 1、房 29(r=8)借 palette 8 → 廊柱/雕飾/穹頂結構色彩正確。
- 使節 14 借 palette 0 → 爬蟲類使節(teal/銅)連貫;使節 16 借 palette 1 → 另一族(橘黃蟲形)連貫。三族各自配對渲染皆乾淨,證明「同 r 配對」。
- 借錯 palette(如房 15 借 palette 0)→ 全畫面雜點。

**先前「DIPLOMAT 解碼損壞」的假設就此推翻**:解碼正常,雜點純粹是**借錯調色盤**;修法是配對律,非改解碼器。

## 三、殘留「穹頂白點」的判讀

frame0 渲染後,穹頂開口區仍有白色抖色點。判定為**原版美術**(穹頂視窗外的抖色星空/星雲,原版會閃爍動畫),非解碼缺陷:
- 該區在房與使節資產都存在、且隨 palette 正確著色(不是隨機噪點)。
- 1996 年 256 色美術以抖色表現星空漸層屬常態。
- 廊柱/使節等實體結構完全乾淨 → 若是解碼 bug 會全域壞,不會只壞穹頂。
