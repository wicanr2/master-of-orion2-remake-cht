# 我們真的把 remake 做完了嗎?——完成度評估報告

> 日期:2026-07-12。方法:對 code / git / 文件實證盤點,不以單元測試綠自評(rulebook/65)。
> 這份報告誠實回答一個問題:**《銀河霸主 II》go/ebiten 繁中重製,到底算不算「完成」了。**

## 一句話結論

**看你用哪把尺量。**

- 以 **CLAUDE.md 列的需求** 為尺 → **實質完成**:每一項指定的交付物都在(可玩 4X + 全繁中 + 音樂 + 忠實新遊戲流程 + 三勝利路徑 + 版本 1.3/1.5 + 致謝 + 中文考據/文化/可行性文件)。
- 以 **「玩起來跟 1996 原版一模一樣」** 為尺 → **未達、也無法只靠自驅達成**:機制層已從手冊/openorion2 錨定,畫面已用原版 LBX 美術,但**還原度%必須由你對原版實測定案**,且幾個系統仍是近似/未對實機核實(見下「誠實缺口」)。

換句話說:**第一版移植完成了;「與原版逐幀等同」是另一個量級的長尾,且卡在需要 DOSBox 實機 oracle 與你的實測判斷。**

## 維度二:可玩性(headless 探針驗證,非自評)

- 玩家 + 3 個性格互異 AI,各自擴張/造艦/外交態勢;60 回合經濟健康(BC 55→419)、無 panic。
- 三條手冊勝利路徑全可達:**征服殲滅 / 銀河議會選舉(忠實搖擺票+棄權)/ 安塔蘭母星反攻**。
- 子系統:殖民地經濟(建築/重力/士氣/礦產/地形/污染/成長/收入/指揮點數)、科技抉擇樹、太空戰(beam/飛彈/球狀分流 + 艦艇佔格 + 武器 mod)、地面戰(陸戰隊/戰車/軌道轟炸)、拓殖、間諜偷科技、領袖技能、AI-vs-AI 關係。

**維度二判定:是一個能從頭玩到分出勝負的遊戲,不是畫面導覽器。**

## 維度三:對原版忠實度(aligned / approximate / unverified)

這是最誠實、也最該保守的一維。

### 已對齊原版(有 oracle 撐腰)
- **畫面美術**:16 畫面載入原版 LBX、擦底疊中文。
- **按鍵像素對齊**:逐畫面比對 openorion2 `initWidgets` 硬編座標,把估計座標補齊到真值(2026-07-12)。無對應 view 的畫面維持 PIL(已達 oracle 上限)。
- **音樂**:原封播放原版 PCM,bit-identical;場景 BGM 切換。
- **數值公式**:14 模組對手冊頁碼 / openorion2 逐條驗證,抓到手冊自身筆誤(AMR/飛彈速度)。
- **經濟校準**:用真存檔 SAVE10.GAM 反推缺失係數(基礎供給、人頭收入),重現手冊記載的開局現象。

### 仍是近似 / 刻意簡化(誠實標注,非 bug)
- 地面戰 d100 核心解算結構沿用一代 1oom 借用結構(force 值用 MOO2 手冊表,但結構未對 MOO2 實機核實)。
- 安塔蘭母星防禦艦隊戰力用保守預設(手冊/openorion2 均無數字)。
- 領袖技能只接 2 個(openorion2 自己也只實作 4/25,是參考碼的天花板)。
- 部分軍事/防禦建築、戰機/航母屬未建的子系統。

### 未經驗證(需外部 oracle)
- **與原版逐幀等同度**:沒有拿原版實機做過側錄比對——這正是「還原度%」不能自評的原因。
- 飛彈速度(手冊公式與附表自相矛盾)、turn-1 母星起始數值細節(如科學家分配)待實機/playtest 定案。

**維度三判定:機制與畫面已錨定原版,但「逐幀等同」未驗證,且少數系統近似。這一維無法只靠自驅收斂到 100%。**

---

## 誠實缺口:是什麼卡住「100% 像原版」

1. **需 DOSBox 實機 oracle** — 飛彈速度、地面戰解算結構、曲目↔場景精確身分、turn-1 數值。這些不臆造,要原版當裁判。
2. **需先建基礎設施** — 戰機/航母(新戰鬥子模型)、部分防禦建築。
3. **設計面 UI 打磨** — 完整 spy/leader/多AI 目標選擇畫面、勝利/落敗結局動畫。
4. **需你的實測判斷** — 還原度%的最終定案,依專案鐵律只能由使用者對原版實測給出。

這四類都不是「還沒動的自驅工作」,而是「卡在外部輸入」——這也是為什麼本輪之後,**沒有清楚定義的純自驅剩項**。

## 最終判定

> **這是一個「需求完成、可玩、機制忠實於手冊」的第一版移植;不是一個「已驗證與 1996 原版逐幀等同」的成品。**

用工程話講:**功能與交付物 done;fidelity 到「手冊/openorion2 錨定」done;fidelity 到「實機驗收」pending your playtest。**
